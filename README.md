# Informatika mintegiko aplikazioa


Aplikazio hau informatika mintegiko inbentarioa kudeatzeko balio du.


## Erabilitako teknologiak
- PHP 8+
- MySQL/MariaDB
- HTML5, CSS3, Bootstrap
- JavaScript (Fetch API, validaciones)


## Funcionalidades principales
- Gestión de usuarios (admin / user)
- Alta, baja y modificación de equipos, aulas y categorías
- Registro de inventario por aula y fecha
- Validación doble: cliente + servidor
- Autenticación por sesiones PHP
